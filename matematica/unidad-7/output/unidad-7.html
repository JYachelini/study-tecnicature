Resumen General

La Unidad 7 cubre dos temas fundamentales de la matemática discreta. Primero, la composición de relaciones (R₂∘R₁): cómo combinar dos relaciones para crear una nueva conexión directa entre conjuntos. Segundo, el Álgebra de Boole: la estructura matemática detrás de la lógica digital, con sus axiomas, leyes de simplificación (Idempotencia, De Morgan, Absorción), Diagramas de Hasse y funciones booleanas con tablas de verdad.

Conceptos Clave

Composición (R₂∘R₁) (z,y)∈R₂∘R₁ si existe x tal que (z,x)∈R₁ y (x,y)∈R₂. Conecta Z con Y eliminando el intermedio X.
Álgebra de Boole Estructura algebraica B=(B,∨,∧,',0,1) con dos elementos mínimos (0 y 1), dos operaciones binarias y una unaria (complemento).
Supremo (∨ o +) Operación binaria similar a la disyunción lógica (OR). x+y vale 1 si al menos uno de los dos vale 1.
Ínfimo (∧ o ·) Operación binaria similar a la conjunción lógica (AND). x·y vale 1 solo si ambos valen 1.
Complemento (') Operación unaria: x'=1 si x=0, y x'=0 si x=1. El opuesto del elemento.
Idempotencia x+x=x y x·x=x. Aplicar la operación al mismo elemento varias veces da el mismo resultado.
Leyes de De Morgan (x+y)'=x'·y' y (x·y)'=x'+y'. El complemento de una suma es el producto de los complementos y viceversa.
Absorción x+(x·y)=x y x·(x+y)=x. Un término "absorbe" al otro si ya lo contiene.
Involución (x')'=x. El doble complemento devuelve el elemento original.
Diagrama de Hasse Representación gráfica de un conjunto parcialmente ordenado finito. Las líneas ascendentes indican el orden entre elementos.
Átomo Elemento minimal justo por encima del 0 en un Diagrama de Hasse. Para todo b, a·b=a o a·b=0.
Función Booleana Función f:Bⁿ→B que toma variables binarias (0 o 1) y devuelve 0 o 1. Su conjunto de verdad V(f) es donde f=1.

Explicación Detallada

Composición de Relaciones

La composición R₂∘R₁ conecta directamente el conjunto Z con el conjunto Y, usando X como intermediario que luego desaparece.

Definición formal: R₂∘R₁ = { (z,y) ∈ Z×Y | ∃x∈X : (z,x)∈R₁ ∧ (x,y)∈R₂ }

Ejemplo con ecuaciones:

R₁: x = z+2    R₂: y = x²

Paso 1: Para z, obtenemos x = z+2 (usando R₁)
Paso 2: Con ese x, obtenemos y = x² = (z+2)² (usando R₂)
Resultado: R₂∘R₁ es la relación y = (z+2)²
Ejemplo con conjuntos:

R₁={(1,2),(2,3)}, R₂={(2,4),(3,6)}

(1,2)∈R₁ y (2,4)∈R₂ → (1,4)∈R₂∘R₁
(2,3)∈R₁ y (3,6)∈R₂ → (2,6)∈R₂∘R₁

R₂∘R₁ = {(1,4),(2,6)}

Axiomas del Álgebra de Boole

AxiomaExpresión con +Expresión con ·
Conmutatividadx + y = y + xx·y = y·x
Distributividadx + (y·z) = (x+y)·(x+z)x·(y+z) = x·y + x·z
Elemento Neutrox + 0 = xx·1 = x
Complementox + x' = 1x·x' = 0

Leyes Derivadas

LeyForma con +Forma con ·
Idempotenciax + x = xx·x = x
Acotaciónx + 1 = 1x·0 = 0
Absorciónx + (x·y) = xx·(x+y) = x
Involución(x')' = x
De Morgan (1)(x+y)' = x'·y'
De Morgan (2)(x·y)' = x'+y'

Simplificación Paso a Paso

Función de control de calidad con sensores A, B, C, D:
F = A'B'C'D + A'B'CD + AB'C'D + AB'CD + AB'CD'

  1. Agrupar: A'B'D(C'+C) + AB'D(C'+C) + AB'CD'
  2. Complemento (x+x'=1): A'B'D·1 + AB'D·1 + AB'CD'
  3. Neutro (x·1=x): A'B'D + AB'D + AB'CD'
  4. Factorizar: B'D(A'+A) + AB'CD'
  5. Complemento: B'D·1 + AB'CD'
  6. Resultado: F = B'D + AB'CD'

Diagramas de Hasse y Átomos

El Diagrama de Hasse representa gráficamente el orden entre los elementos de un álgebra de Boole finita. Las líneas ascendentes indican inclusión o precedencia.

Átomo: un elemento a es un átomo si para cualquier b del álgebra, se cumple que a·b=a o a·b=0 (no hay nada "entre" el 0 y a).

Ejemplo: En el conjunto de partes P({a,b,c}), los átomos son {a}, {b} y {c} (los conjuntos unitarios). Están justo por encima del ∅ en el diagrama.
