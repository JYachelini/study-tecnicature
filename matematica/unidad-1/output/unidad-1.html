Overview de qué trata la Unidad 1 — extraído del material de cátedra

¿De qué trata esta unidad?

La Unidad 1 de Matemática 1 introduce la Teoría de Conjuntos como fundamento del álgebra moderna. Un conjunto es una colección bien definida de objetos llamados elementos; queda determinado por sus elementos, sin importar el orden ni la repetición.

"Un conjunto queda determinado por sus elementos. Dos conjuntos son iguales si y solo si tienen exactamente los mismos elementos."

Se estudian las dos relaciones fundamentales: la pertenencia (∈), que vincula un elemento con un conjunto, y la inclusión (⊆), que vincula dos conjuntos. La distinción entre ambas es fuente de los errores más frecuentes en los ejercicios.

Las operaciones entre conjuntos —unión (∪), intersección (∩), diferencia (∖), diferencia simétrica (△) y complemento (ᶜ)— permiten construir nuevos conjuntos. El conjunto de partes P(A) reúne todos los subconjuntos de A: si A tiene n elementos, P(A) tiene 2ⁿ elementos. Los diagramas de Venn permiten visualizar estas relaciones gráficamente.

Conjunto
Colección bien definida de objetos llamados elementos. Queda determinado únicamente por sus elementos, no por el orden ni la repetición.
A = {1, 2, 3} · B = {a, b, c}

e
Elemento
Cada uno de los objetos que componen un conjunto. Un elemento puede ser cualquier objeto matemático: número, símbolo, otro conjunto.
En A = {1, 2, {3}}: los elementos son 1, 2 y {3}
03 / 14
∈
Pertenencia
a ∈ A significa "el objeto a es un elemento del conjunto A". Relaciona un objeto con un conjunto. Lo opuesto es ∉ (no pertenece).
1 ∈ {1,2,3} → Verdadero {1} ∈ {1,2,3} → Falso
04 / 14
⊆
Subconjunto (Inclusión)
A ⊆ B si todo elemento de A también es elemento de B. Relaciona dos conjuntos. Todo conjunto es subconjunto de sí mismo: A ⊆ A.
{1,2} ⊆ {1,2,3} → Verdadero {4} ⊆ {1,2,3} → Falso
05 / 14
⊂
Subconjunto Propio
A ⊂ B si A ⊆ B y A ≠ B. Es decir, A está contenido en B pero B tiene al menos un elemento que no está en A.
{1,2} ⊂ {1,2,3} → Verdadero {1,2,3} ⊂ {1,2,3} → Falso
06 / 14
∅
Conjunto Vacío
El conjunto sin elementos, denotado ∅ o {}. Propiedad clave: ∅ ⊆ A para cualquier conjunto A. Pero ∅ ∈ A solo si ∅ figura como elemento explícito.
∅ ⊆ {1,2,3} → Verdadero (siempre) ∅ ∈ {1,2,3} → Falso
07 / 14
U
Conjunto Universal
El conjunto referencial U que contiene todos los elementos del contexto del problema. Todos los conjuntos son subconjuntos de U.
U = {1, {3}, -2, 7, 10, {1,2,3}, 3}
08 / 14
∪
Unión
A ∪ B = {x : x ∈ A ó x ∈ B}. Contiene todos los elementos de A, todos los de B, o de ambos. La unión siempre es al menos tan grande como cada uno.
A={1,2,3}, B={3,4,5} A ∪ B = {1,2,3,4,5}
09 / 14
∩
Intersección
A ∩ B = {x : x ∈ A y x ∈ B}. Contiene solo los elementos comunes a ambos conjuntos. Si no hay elementos en común, A ∩ B = ∅ (conjuntos disjuntos).
A={2,3,5,7,8}, B={3,5,1} A ∩ B = {3,5}
10 / 14
∖
Diferencia
A ∖ B = {x : x ∈ A y x ∉ B}. Elementos de A que no pertenecen a B. El orden importa: A ∖ B ≠ B ∖ A en general.
A={1,5,7}, B={7,3,4,2,1} B ∖ A = {3,4,2}
11 / 14
Aᶜ
Complemento
Aᶜ = {x ∈ U : x ∉ A}. El conjunto de todos los elementos del universo U que no pertenecen a A. Equivale a U ∖ A.
Aᶜ = U ∖ A (A ∩ Aᶜ) = ∅ y (A ∪ Aᶜ) = U
12 / 14
△
Diferencia Simétrica
A △ B = (A ∖ B) ∪ (B ∖ A). Los elementos que pertenecen a uno u otro pero no a ambos simultáneamente. Es conmutativa: A △ B = B △ A.
A={1,-2,7,3}, B={1,{3},10} A △ B = {-2,7,3,{3},10}
13 / 14
P(A)
Conjunto de Partes
P(A) es el conjunto de todos los subconjuntos de A, incluyendo ∅ y el propio A. Si |A| = n, entonces |P(A)| = 2ⁿ.
A = {a,b}: P(A) = {∅,{a},{b},{a,b}} |P(A)| = 2² = 4
14 / 14
⊙
Diagrama de Venn
Representación gráfica de conjuntos como curvas (generalmente círculos) dentro de un rectángulo que representa U. Las áreas de solapamiento representan intersecciones.
Útil para visualizar unión, intersección, diferencia y complemento

A Definición y Formas de Representar un Conjunto

Un conjunto es una colección bien definida de objetos. "Bien definida" significa que para cualquier objeto, se puede determinar con certeza si pertenece o no al conjunto.

Existen tres formas principales de definir un conjunto:

Por Extensión
Listando todos los elementos
A = {1, 2, 3, 4, 5}
Por Comprensión
Describiendo la propiedad
A = {x ∈ ℕ : x ≤ 5}
Por Diagrama
Representación de Venn
Círculos que engloban elementos visualmente
Conjuntos Numéricos Notables: ℕ = {0, 1, 2, 3, ...} Naturales ℤ = {..., -2, -1, 0, 1, 2, ...} Enteros ℚ = {p/q : p,q ∈ ℤ, q ≠ 0} Racionales ℝ = todos los números reales Reales Inclusión: ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ

∈ Pertenencia (∈) vs Inclusión (⊆)

Esta es la distinción más importante y la fuente de errores más frecuente. ∈ relaciona un elemento con un conjunto. ⊆ relaciona dos conjuntos entre sí.

Dado A = {1, 2, {3}, {1,2}}: 1 ∈ A → Verdadero (1 es elemento de A) {3} ∈ A → Verdadero ({3} figura como elemento de A) 3 ∈ A → Falso (3 no está; sí lo está {3}) {3} ⊆ A → Falso (requeriría que 3 ∈ A, que es falso) {{3}} ⊆ A → Verdadero ({3} ∈ A, entonces {{3}} ⊆ A) {1,2} ∈ A → Verdadero ({1,2} figura como elemento de A) {1,2} ⊆ A → Verdadero (1 ∈ A y 2 ∈ A) ∅ ⊆ A → Verdadero (vacío es subconjunto de todo) ∅ ∈ A → Falso (∅ no figura como elemento de A)
Ejercicio 9 resuelto — de la práctica
A = {1},  B = {{1}}

a) 1 ∈ A    ✔  (1 es el único elemento de A)
b) 1 ∈ B    ✘  (el único elemento de B es {1}, no 1)
c) {1} ⊆ A  ✔  (1 ∈ A, por lo tanto {1} ⊆ A)
d) {1} ∈ B  ✔  ({1} es exactamente el único elemento de B)
e) {{1}} ⊆ A ✘  (para esto, {1} debería ∈ A, pero A = {1} y {1} ∉ {1})

Regla práctica: si ves el símbolo ∈, preguntá "¿es esto un objeto que figura como miembro de ese conjunto?". Si ves ⊆, preguntá "¿todos los miembros del conjunto de la izquierda están en el de la derecha?".

∪ Operaciones: Unión, Intersección y Diferencia

Estas tres operaciones permiten construir nuevos conjuntos a partir de otros. Veamos ejercicios resueltos de la práctica:

Ejercicio 14 resuelto — Unión, Intersección, Diferencia
A = {2,3,5,7,8},  B = {3,5,1},  C = {7,9},  D = {9,4,1}

a) (A ∩ B) ∪ (C ∩ D)
   A ∩ B = {3,5}           C ∩ D = {9}
   Resultado: {3,5} ∪ {9} = {3,5,9}

b) (B ∪ C) ∩ A
   B ∪ C = {3,5,1,7,9}
   Resultado: {3,5,1,7,9} ∩ {2,3,5,7,8} = {3,5,7}

c) (A ∩ D) ∪ A ∪ B
   A ∩ D = ∅  → ∅ ∪ A = A
   Resultado: {2,3,5,7,8} ∪ {3,5,1} = {2,3,5,7,8,1}
Ejercicio 15 resuelto — Diferencia de conjuntos
A = {1,5,7},  B = {7,3,4,2,1},  C = {5},  D = {2,9,7}

a) (B ∖ A) ∪ C
   B ∖ A = {3,4,2}     Resultado: {3,4,2} ∪ {5} = {3,4,2,5}

b) (B ∖ A) ∖ (C ∪ D)
   C ∪ D = {5,2,9,7}   Resultado: {3,4,2} ∖ {5,2,9,7} = {3,4}

c) (A ∪ B) ∖ (D ∪ C)
   A ∪ B = {1,5,7,3,4,2}
   D ∪ C = {2,9,7,5}   Resultado: {1,5,7,3,4,2} ∖ {2,9,7,5} = {1,3,4}

d) A ∪ (B ∖ D)
   B ∖ D = {3,4,1}     Resultado: {1,5,7} ∪ {3,4,1} = {1,5,7,3,4}

△ Diferencia Simétrica y Complemento

La diferencia simétrica A △ B contiene los elementos que están en exactamente uno de los dos conjuntos. Para el complemento, siempre se necesita conocer el conjunto universal U.

Ejercicio 16 resuelto — Diferencia simétrica y complemento
A = {1,-2,7,3},  B = {1,{3},10},  C = {-2,{1,2,3},3}
U = {1,{3},-2,7,10,{1,2,3},3}

a) A ∩ (B △ C)
   B △ C = {1,{3},10,-2,{1,2,3},3}   (todos, ya que no hay comunes)
   Resultado: A ∩ (B △ C) = {1,-2,3}

b) (A ∩ B) △ (A ∩ C)
   A ∩ B = {1}
   A ∩ C = {-2,3}
   {1} △ {-2,3} = {1,-2,3}   (son disjuntos → su △ es su unión)

c) Aᶜ ∩ Bᶜ ∩ Cᶜ = (A ∪ B ∪ C)ᶜ   (De Morgan)
   A ∪ B ∪ C = {1,-2,7,3,{3},10,{1,2,3}}
   (A ∪ B ∪ C)ᶜ = U ∖ (A ∪ B ∪ C) = ∅  (no quedan elementos)

P Conjunto de Partes P(A)

P(A) es el conjunto de todos los subconjuntos de A, incluyendo el vacío y el propio A. Si |A| = n, entonces |P(A)| = 2ⁿ.

Ejercicio 18 — Hallar P(A): a) A = {1} P(A) = {∅, {1}} |P(A)| = 2¹ = 2 b) A = {a, b} P(A) = {∅, {a}, {b}, {a,b}} |P(A)| = 2² = 4 c) A = {1, {1,2}, 3} P(A) = {∅, {1}, {{1,2}}, {3}, {1,{1,2}}, {1,3}, {{1,2},3}, A} |P(A)| = 2³ = 8 d) A = ∅ P(∅) = {∅} |P(∅)| = 2⁰ = 1

Nota importante: en c), el elemento del conjunto es {1,2} (un conjunto), por lo que al listar subconjuntos de A, ese elemento aparece encerrado en llaves: {{1,2}}.
