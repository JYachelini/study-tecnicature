Teoría de Conjuntos

Fundamentos, operaciones y propiedades de los conjuntos

📁 Material: Práctica 1 Ejercicios Resueltos Clase 1 – Conjuntos Operaciones

Resumen General

La Teoría de Conjuntos es la base del pensamiento matemático. Un conjunto es una colección de objetos bien definidos llamados elementos, identificados por la relación de pertenencia (∈). Los conjuntos se pueden definir por extensión (listando elementos: A = {a, e, i, o, u}) o por comprensión (describiendo su propiedad: B = {x | x es vocal}).

Se trabajan conjuntos especiales como el vacío (∅), el Universal (U) y el Conjunto de Partes P(A). Las operaciones fundamentales son: unión (∪), intersección (∩), diferencia (−), diferencia simétrica (△) y complemento (′). Estas cumplen las propiedades del Álgebra de Boole, incluyendo las célebres Leyes de De Morgan. Los Diagramas de Venn son la herramienta gráfica clave para visualizar y resolver ejercicios.

Conceptos Clave

∈
Conjunto y Pertenencia
Colección de objetos bien definidos y diferenciables llamados elementos. Si a pertenece a A se escribe a ∈ A; si no, a ∉ A.
A = {1, 2, 3} → 2 ∈ A pero 5 ∉ A
{ }
Extensión y Comprensión
Por extensión: se listan todos los elementos. Por comprensión: se enuncia la propiedad característica.
Extensión: A = {a,e,i,o,u} · Comprensión: A = {x | x es vocal}
∅
Conjunto Vacío
El conjunto que no contiene ningún elemento. Está contenido en cualquier conjunto: ∅ ⊆ A para todo A.
P
Conjunto de Partes P(A)
El conjunto formado por todos los subconjuntos posibles de A, incluyendo ∅ y A. Si |A| = n, entonces |P(A)| = 2ⁿ.
A = {a,b} → P(A) = {∅, {a}, {b}, {a,b}} → |P(A)| = 4
⊆
Inclusión (Subconjunto)
A ⊆ B si todo elemento de A también pertenece a B. A = B si y solo si A ⊆ B y B ⊆ A simultáneamente.
ℕ ⊆ ℤ ⊆ ℚ ⊆ ℝ ⊆ ℂ
∪
Unión
Todos los elementos que pertenecen a A, a B, o a ambos.
{1,2} ∪ {2,3} = {1,2,3}
∩
Intersección
Solo los elementos que pertenecen simultáneamente a A y a B.
{1,2,3} ∩ {2,3,4} = {2,3}
−
Diferencia (A − B)
Elementos de A que NO están en B. Fórmula: |A−B| = |A| − |A∩B|
{1,2,3} − {2,3,4} = {1}
△
Diferencia Simétrica
Elementos en A o en B, pero NO en ambos. A△B = (A−B) ∪ (B−A) = (A∪B) − (A∩B)
{1,2,3} △ {2,3,4} = {1,4}
′
Complemento (A′)
Elementos del Universal que NO pertenecen a A. A ∪ A′ = U y A ∩ A′ = ∅.
U={1..5}, A={1,2,3} → A′={4,5}
⟺
Leyes de De Morgan
(A ∪ B)′ = A′ ∩ B′ · (A ∩ B)′ = A′ ∪ B′
Permiten transformar complementos de operaciones compuestas

Explicación Detallada

1. Pertenencia vs. Inclusión

La distinción más importante y fuente de errores frecuentes: pertenencia (∈) e inclusión (⊆) son conceptos completamente distintos.

Dado A = {1, 2, {3}, {1,2}}: • 3 ∈ A → FALSO (3 no está; {3} sí está) • {3} ⊆ A → FALSO ({3} es elemento de A, no subconjunto) • {3} ∈ A → VERDADERO • {1,2} ∈ A → VERDADERO ({1,2} es un elemento de A) • {1,2} ⊆ A → FALSO (A no contiene 1 y 2 como elementos directos)

2. Las Cinco Operaciones

∪
Unión
A o B (o ambos)
∩
Intersección
A y B a la vez
−
Diferencia
A pero no B
△
Dif. Simétrica
Uno u otro, no ambos
′
Complemento
Lo que falta en U

3. Conjunto de Partes P(A)

P(A) contiene todos los posibles subconjuntos de A, incluyendo ∅ y el propio A. Si |A| = n → |P(A)| = 2ⁿ.

A = {a, b, c} P(A) = { ∅, {a}, {b}, {c}, {a,b}, {a,c}, {b,c}, {a,b,c} } |P(A)| = 2³ = 8

4. Leyes de De Morgan (verificación numérica)

U = {1,2,3,4}, A = {1,2}, B = {2,3} (A∪B)′: A∪B = {1,2,3} → complemento = {4} A′ ∩ B′: A′={3,4}, B′={1,4} → A′∩B′ = {4} ✓ (A∩B)′: A∩B = {2} → complemento = {1,3,4} A′ ∪ B′: {3,4} ∪ {1,4} = {1,3,4} ✓

5. Propiedades del Álgebra de Boole

Idempotencia: A ∪ A = A · A ∩ A = A Conmutativa: A ∪ B = B ∪ A Asociativa: A ∪ (B ∪ C) = (A ∪ B) ∪ C Absorción: A ∪ (A ∩ B) = A Complemento: A ∪ A′ = U · A ∩ A′ = ∅

6. Técnica para Diagramas de Venn

Para expresiones tipo (A ∪ B′) ∩ C se resuelve de adentro hacia afuera, respetando los paréntesis.

Resolver (A ∪ Bᶜ) ∩ C: Paso 1: Calcular Bᶜ (complemento de B) Paso 2: Calcular A ∪ Bᶜ Paso 3: Intersectar con C
