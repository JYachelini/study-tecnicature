Estructuras de Repetición

Mientras (While) · Para (For) · Repetir...Hasta (Do-While) · Contadores y Acumuladores

Resumen General

Las estructuras de repetición (también llamadas ciclos o bucles) permiten ejecutar un bloque de instrucciones más de una vez. Sin ellas, habría que repetir manualmente el código N veces. Se estudian tres ciclos: Mientras (While), Para (For) y Repetir...Hasta (Do-While). También se trabaja con contadores (cuentan iteraciones) y acumuladores (suman valores).

Mientras (While)
Repite mientras la condición sea Verdadera. Puede no ejecutarse nunca.
Cuando no sé cuántas veces repetir.
Para (For)
Repite un número conocido de veces. Ideal cuando se sabe la cantidad exacta.
Cuando sé exactamente cuántas veces repetir.
Repetir...Hasta (Do-While)
Ejecuta al menos una vez. Repite HASTA que la condición sea Verdadera.
Cuando necesito ejecutar al menos una vez.

Conceptos Clave

Ciclo (Bucle)
Estructura que repite un bloque de instrucciones mientras se cumpla una condición.
Iteración
Cada ejecución del cuerpo del ciclo. 10 iteraciones = el bloque se ejecutó 10 veces.
Contador
Variable que se incrementa en cada iteración para contar cuántas veces se ejecutó el ciclo. Inicializado en 0 (o 1).
Acumulador
Variable que suma valores en cada iteración. Inicializado en 0.
Mientras (While)
Evalúa la condición ANTES de cada iteración. Puede no ejecutarse si la condición es Falsa desde el inicio.
Para (For)
Incluye inicialización, condición e incremento en una sola línea. Ideal para rangos conocidos.
Repetir...Hasta
Evalúa la condición DESPUÉS de cada iteración. Se ejecuta al menos una vez.
Bucle Infinito
Ciclo cuya condición nunca se vuelve Falsa. Es un error de programación (salvo casos específicos).

Explicación Detallada

¿Por qué necesitamos ciclos?

Sin ciclos, para hacer la misma operación con 10 personas habría que repetir el código 10 veces. Con un ciclo, basta con escribirlo una vez y decirle cuántas veces repetirlo.

Mientras (While)

Evalúa la condición antes de cada iteración. Si es Falsa desde el principio, el bloque nunca se ejecuta.

// Pseudocódigo: Mientras (contador < 10) Hacer Mostrar "Ingrese la edad: "; Leer edad; Si ( edad >= 18) Entonces Mostrar "Es mayor de edad"; Sino Mostrar "Es menor de edad"; Fin Si contador = contador + 1; FinMientras // En C++: while (contador < 10) { // ... instrucciones ... contador++; }

Para (For)

Concentra inicialización, condición e incremento en una sola línea. Ideal cuando se conoce la cantidad de iteraciones.

// Pseudocódigo: Para i desde 1 hasta 5 Hacer Mostrar "Hola"; FinPara // En C++: for (int i = 1; i <= 5; i++) { cout << "Hola"; }

Repetir...Hasta (Do-While)

Ejecuta el bloque al menos una vez y evalúa la condición al final de cada iteración.

// Pseudocódigo: Repetir Mostrar "Ingrese un número positivo: "; Leer num; Hasta Que ( num > 0 ) // En C++: do { cout << "Ingrese un número positivo: "; cin >> num; } while (num <= 0);
