Resumen General

Las estructuras de control permiten que un programa tome distintos caminos según se cumplan ciertas condiciones. Las condiciones son expresiones lógicas que arrojan Verdadero o Falso. Se estudian tres estructuras: Si...Entonces (una rama), Si...Entonces...Sino (dos ramas) y Según sea (múltiples opciones). También se estudian condicionales anidadas.

Las 3 estructuras de control de selección

  • Si...Entonces: ejecuta acciones solo si la condición es Verdadera.
  • Si...Entonces...Sino: ejecuta un bloque si es Verdadera, otro si es Falsa.
  • Según sea: evalúa una variable y ejecuta un bloque según su valor (como switch).

Conceptos Clave

Estructura de Control
Instrucción que controla el flujo de ejecución del programa según condiciones. El programa puede "tomar decisiones".
Condición
Expresión lógica que evalúa a Verdadero o Falso. Ej: (nota >= 6), (edad > 18).
Si...Entonces
Si la condición es Verdadera, ejecuta las acciones del bloque. Si es Falsa, no ejecuta nada.
Si...Entonces...Sino
Si la condición es Verdadera ejecuta el bloque "Entonces". Si es Falsa ejecuta el bloque "Sino".
Según sea (Switch)
Evalúa el valor de una variable y ejecuta el bloque correspondiente a cada caso posible.
Condicional Anidada
Una estructura Si...Entonces dentro de otra estructura Si...Entonces. Permite evaluar múltiples condiciones en cascada.
Flujo de Ejecución
El orden en que se ejecutan las instrucciones del programa. Las estructuras de control lo modifican.
Operador &&, ||, !
En C/C++: && (Y lógico), || (O lógico), ! (NO lógico). Combinan condiciones.

Explicación Detallada

Si...Entonces (una rama)

Evalúa la condición. Si es Verdadera, ejecuta las acciones del bloque. Si es Falsa, las ignora y continúa.

Si ( numero > 0 ) Entonces Mostrar "Número mayor a 0"; Fin Si // En C++: if (numero > 0) { cout << "Número mayor a 0"; }

Si...Entonces...Sino (dos ramas)

Si la condición es Verdadera ejecuta el bloque "Entonces". Si es Falsa, ejecuta el bloque "Sino". Siempre se ejecuta uno u otro.

Si ( nota >= 6 ) Entonces Mostrar "Materia aprobada"; Sino Mostrar "Materia desaprobada"; Fin Si // En C++: if (nota >= 6) { cout << "Materia aprobada"; } else { cout << "Materia desaprobada"; }

Condicionales Anidadas

Permiten evaluar múltiples rangos de valores. Cada "Sino" puede contener otro "Si...Entonces".

Si ( promedio < 4 ) Entonces Mostrar "Insuficiente"; Sino Si ( promedio < 6 ) Entonces Mostrar "Regular"; Sino Si ( promedio < 8 ) Entonces Mostrar "Bien"; Sino Mostrar "Muy Bien / Sobresaliente"; Fin Si Fin Si Fin Si

Condición combinada

Un número es "de la suerte" si: es positivo Y impar Y múltiplo de 3, O si es 20 o 80.

(num > 0 && num % 2 != 0 && num % 3 == 0) || (num == 20 || num == 80)
